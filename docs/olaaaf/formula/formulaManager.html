Module olaaaf.formula.formulaManager

This class allows the user to easily declare a new Formula thanks to FormulaManager.parser() and store in memory all previously named formulas, either via their constructor or the function FormulaManager.declare().

def parser(string: str)

Function allowing the user to intuitively parse a Formula from a string, using an infixed notation and customizable operators.

While more intuitive due to the less restrictive scope of usable operators, this method of declaring formulas assume you previously named them, either via the fmName attribute in their constructor or thanks to FormulaManager.declare(). The operators could be customized in Constants but are by default:

  • & for the and operator, represented by And

  • | for the or operator, represented by Or

  • ~ for the not operator, represented by Not

  • -> for the implication operator, represented by Implication

  • <-> for the equivalence operator, represented by Equivalence

  • <+> for the xor operator, represented by Xor
