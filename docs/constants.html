var ONLY_ONE_SOLUTION

If set to True, the revision algorithm will by default only return one point that satisfies \psi \circ \mu. If not, it will return all solutions. By default, this constant is set to True.

Note that this only change the default value of the onlyOneSolution parameter of Revision, it could still be changed on a case-by-case basis for every new instance of this class.
